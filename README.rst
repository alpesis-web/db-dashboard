Lucky Dog is a financial analytics platform.

==============================================================================
Getting Started
==============================================================================

::

    $ pip3 install requirements/base.txt
    $ cd dashboard
    $ python3 manage.py migrate
    $ python3 manage.py runserver
    # open the website at http://localhost:8000
